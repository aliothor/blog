# 🧊 数据标准


[GeoJSON](https://stevage.github.io/geojson-spec/) : GeoJSON specification

[GeoArrow](https://github.com/geoarrow/geoarrow) : Specifications for storing geospatial data in Apache Arrow and Apache Parquet

[ShapeFile](https://www.esri.com/content/dam/esrisites/sitecore-archive/Files/Pdfs/library/whitepapers/pdfs/shapefile.pdf) : Esri ShapeFile

[GeoPackage](https://www.ogc.org/standard/geopackage/) : GeoPackage Encoding Standard

[Vector Tile](https://github.com/mapbox/vector-tile-spec) : Mapbox Vector Tile Specification

[COG](https://www.cogeo.org/) : A Cloud Optimized GeoTIFF (COG) is a regular GeoTIFF file, aimed at being hosted on a HTTP file server, with an internal organization that enables more efficient workflows on the cloud.

[3d-tiles](https://github.com/CesiumGS/3d-tiles) : Specification for streaming massive heterogeneous 3D geospatial datasets
